# Get Trending

| Parameter | Description                         |
| --------- | ----------------------------------- |
| `time`    | day/week/month/3_month/6_month/year |
| `page`    | page limit may vary                 |

Example request:

```
GET https://hanime-api-five.vercel.app/trending/day/1
```
